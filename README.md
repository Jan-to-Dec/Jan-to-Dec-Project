# 🚇 Traffic: 서울 지하철 승하차 데이터 분석

본 프로젝트는 서울 지하철 노선의 승하차 데이터를 분석하여 유의미한 인사이트를 도출하는 것을 목표로 합니다. 시간대별, 역별 승하차 패턴을 파악하고 클러스터링을 통해 역들의 특성을 분류했습니다.

### **데이터 소개**

**데이터 출처**: 서울시 지하철 호선별 역별 시간대별 승하차 인원 정보 (https://data.seoul.go.kr/dataList/OA-12252/S/1/datasetView.do)

**주요 변수**: 사용월, 연도, 월, 지하철역, 시간대별 승차인원

### **고급 분석**

# Insight

#### **1. 승차인원 상위 역**

상위 5개 역의 월평균 승차인원

1. 강남: 2,627,498명
2. 잠실: 2,334,455명
3. 홍대입구: 2,057,443명
4. 신림: 1,906,915명
5. 구로디지털단지: 1,731,064명

#### **2. 연도별 승차인원 추이**
2015년부터 2019년까지 꾸준히 증가하는 추세를 보였습니다.
2020년에 급격한 감소가 있었으며, 이는 COVID-19 팬데믹의 영향으로 추정됩니다.
2021년부터 점진적인 회복세를 보이고 있습니다.

#### **3. 시간대별 패턴**
대부분의 역에서 오전 8-9시, 오후 6-7시에 피크가 관찰됩니다.
강남, 삼성 등 업무지구 역들은 오후 피크가 더 높게 나타납니다.

#### **4. 역 클러스터링 결과**
K-means 클러스터링 결과, 3개의 뚜렷한 클러스터가 발견되었습니다:
**클러스터 0 (업무지구 중심)**: 강남, 삼성, 역삼, 을지로입구 등
**클러스터 1 (주거지역 중심)**: 구로디지털단지, 신림, 서울대입구 등
**클러스터 2 (복합지역)**: 잠실, 홍대입구, 건대입구 등

#### **결론**

2호선은 서울의 주요 업무 및 상업지구를 연결하는 노선이라는 사실이 명확합니다. 
COVID-19 팬데믹의 영향이 뚜렷하게 관찰되며, 지속적인 회복 추세 모니터링이 필요합니다.
역별 특성에 맞는 맞춤형 서비스 제공을 고려해볼 수 있습니다. 
향후에는 날씨, 주요 이벤트 등 외부 요인과 관련지어서 연관성 분석을 할 수 있습니다.

# 🌐 Walmart 매장 판매 예측: 데이터 기반 인사이트와 머신러닝 모델링

## **프로젝트의 배경 및 목표**

이 프로젝트는 Walmart의 매장 판매 데이터를 분석하여 매출 예측 모델을 개발하는 것을 목표로 합니다. Walmart와 같은 대형 소매업체에서는 정확한 매출 예측이 재고 관리, 인력 배치, 마케팅 전략 수립 등에 중요한 역할을 합니다. 이 프로젝트를 통해 다양한 요인들이 매출에 미치는 영향을 파악하고, 이를 바탕으로 더 효율적인 비즈니스 운영 방안을 모색하고자 합니다.

## **활용한 데이터셋**

이 프로젝트에서는 다음과 같은 데이터셋을 활용했습니다:

1. stores.csv: 각 매장의 정보 (유형, 크기 등)
2. train.csv: 2010-02-05부터 2012-11-01까지의 주간 매출 데이터
3. test.csv: 2012-11-02부터 2013-07-26까지의 매출 예측을 위한 테스트 데이터
4. features.csv: 매장별, 날짜별 추가 특성 (온도, 연료 가격, 마크다운 등)

## **프로젝트 진행 과정**

1. 데이터 로딩 및 병합: 여러 CSV 파일의 데이터를 불러와 하나의 데이터프레임으로 병합했습니다.
2. 데이터 전처리: 날짜 데이터를 연도, 월, 일, 주 단위로 분리, 결측치 확인 및 처리
3. 탐색적 데이터 분석 (EDA): 
    
    매장 유형별 인기도 및 평균 매출 분석
    
    연도별, 월별, 주별 매출 트렌드 분석
    
    부서별 평균 매출 분석
    
    공휴일과 비공휴일의 매출 비교
    
4. 시각화: matplotlib, seaborn, plotly 등을 활용하여 다양한 차트와 그래프 생성

## **분석 결과**

1. 매장 유형별 분석:
    
    A 타입 매장이 전체의 85.2%로 가장 많았으며, B 타입(9.3%)과 C 타입(5.5%)이 뒤를 이었습니다.
    
    B 타입 매장의 평균 매출이 가장 높았고, A 타입과 C 타입이 그 뒤를 이었습니다.
    
2. 시간별 매출 트렌드:
    
    연도별 월간 매출을 비교한 결과, 11월과 12월에 매출이 가장 높았습니다.
    
    주간 매출 분석 결과, 추수감사절과 크리스마스 전 주에 매출이 크게 증가했습니다.
    
3. 부서별 매출:
    
    부서별 평균 매출에 큰 차이가 있었으며, 38, 65, 72, 92, 95번 부서의 매출이 특히 높았습니다.
    
4. 공휴일 영향:
    공휴일의 평균 매출이 비공휴일보다 높게 나타났습니다.
    

## **프로젝트를 통해 얻은 인사이트**

계절성: 연말(11월, 12월)에 매출이 크게 증가하는 뚜렷한 계절성을 보입니다. 이 시기에 맞춘 특별 프로모션이나 재고 관리가 중요할 것으로 보입니다.
매장 유형별 전략: B 타입 매장의 평균 매출이 가장 높으므로, 이 유형의 매장 특성을 분석하여 다른 유형의 매장에도 적용할 수 있는 전략을 개발할 필요가 있습니다.
부서별 차별화: 매출이 높은 부서(38, 65, 72, 92, 95)의 특성을 분석하여 다른 부서의 매출 향상 전략을 수립할 수 있습니다.
공휴일 마케팅: 공휴일의 매출이 더 높으므로, 공휴일 특별 프로모션이나 마케팅 전략을 강화하는 것이 효과적일 것입니다.
데이터 기반 의사결정: 이러한 분석 결과를 바탕으로 재고 관리, 인력 배치, 마케팅 전략 등을 최적화할 수 있을 것으로 보입니다.
이 프로젝트를 통해 Walmart의 매출에 영향을 미치는 다양한 요인들을 파악할 수 있었으며, 이는 향후 더 정확한 매출 예측 모델 개발의 기초가 될 것입니다.
